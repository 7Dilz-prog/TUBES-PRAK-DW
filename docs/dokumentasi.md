## Workflow Branch - na
1. Masuk terminal kemudian gunakan gitbash
2. Kondisi branch sekarang di branch utama "Delsyad"
3. Buat branch dulu --> git checkout -b NAMAKALIAN(nama panggilan aja). contoh : git checkout -b Elin
4. Setelah selesai ngerjain --> git add .
5. git commit -m "misal: fixed bug y'all"
6. git push -u origin NAMA-BRANCH-KALIAN. contoh: git push -u origin Elin
7. Pas mau ngerjain lagi -> git pull origin Delsyad
8. Masuk ke branch masing-masing
9. git merge origin Delsyad
10. Lakukan poin 2 sampai 4
