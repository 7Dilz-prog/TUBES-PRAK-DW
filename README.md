# Workflow Branch
1. Masuk terminal kemudian gunakan gitbash
1. Buat branch dulu --> git checkout -b NAMAKALIAN(nama panggilan aja). contoh : git checkout -b Elin
2. git add .
3. git commit -m "misal: fixed bug y'all"
4. git push -u origin NAMA-BRANCH-KALIAN. contoh: git push -u origin Elin
5. Pas mau ngerjain lagi -> git pull origin Delsyad
6. Masuk ke branch masing-masing
7. git merge origin Delsyad
8. Lakukan poin 2 sampai 4
